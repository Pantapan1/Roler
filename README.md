# Структура проекта

Бот разбит на отдельные файлы (без подпапок), по одному файлу на команду/обработчик:

- `shared.py` — общий код: подключение к БД, класс Session, FSM-состояния,
  вспомогательные функции (roll_dice, генерация картинок и т.д.), инициализация
  bot/dp и всех роутеров. Все остальные файлы делают `from shared import *`.
- `cmd_<название>.py` — один файл = один обработчик команды (например
  `cmd_roll.py` — команда /roll, `cmd_market.py` — команда /market и т.д.)
- `main.py` — точка входа: импортирует все `cmd_*.py` (чтобы обработчики
  зарегистрировались в роутерах) и запускает поллинг.

## Запуск

```
pip install -r requirements.txt
python main.py
```

Важно: не переименовывайте и не удаляйте `shared.py` — от него зависят все
остальные файлы. Если добавляете новую команду — создайте новый `cmd_*.py`
файл (по образцу соседних) и добавьте `import cmd_ваша_команда` в `main.py`.
